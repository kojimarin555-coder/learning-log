# Day 03
## 今天学了什么
-拆位数：用 % 10 取余数、// 10 取整数部分这两个基本运算
-循环里边找边判断，找到就 break：用一个布尔变量（found）在循环外先设成 False，
循环内一旦条件成立就打印、把 found 改成 True、然后 break 跳出循环；循环结束后用 if not found: 判断到底有没有找到，没找到就提示一句。

## 两个基础概念：
-类型转换（int() / str() / float() 这类，把值从一种类型转成另一种）
  -os：虽然这次题目里其实用不上。
-if 变量: 的判断逻辑——不是"看到变量名字就执行"，而是看这个变量当前存的值是不是 True。

## 卡点
-一开始把整个 for 循环塞进了 if found: 里面：
  -因为一开始把 found 的初始值搞反了，先写成 found = True，然后又把整个 for 循环缩进放进了 if found: 下面。
    里面的 for 循环根本没机会运行——不是循环本身有 bug，循环根本没跑起来，这个问题绕了很久。。
-没搞清楚 found 这个变量到底该不该控制 for 循环：后来明白了，found 只是一个"记录本"，用来标记循环内部有没有找到目标，for 循环本身应该正常执行，不需要被任何 if 包住。
